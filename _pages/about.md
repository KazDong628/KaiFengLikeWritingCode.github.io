**MatchStruct（赛理）** 是面向足球竞赛的**赛果结构研究与复盘平台**。平台以可审计的分层口径，展示赛前情景推演、赛后覆盖复盘与方法论说明，帮助理解比赛结果、净胜幅度、进球区间与比分情景之间的关系。

## 1. 产品定位

| 维度 | 说明 |
| --- | --- |
| 产品名称 | MatchStruct / 赛理 |
| 产品类型 | 体育数据分析研究演示平台（静态信息展示） |
| 核心能力 | 分层复盘、情景推演、方法透明、合规提示 |
| 不适用场景 | 购彩指导、投注建议、收益承诺、资金往来 |

## 2. 我们提供 / 不提供

| 我们提供 | 我们不提供 |
| --- | --- |
| 基于公开信息的结构分析与情景覆盖 | 投注建议、购彩指导、跟单或代购 |
| 分层复盘与模型评估口径 | “必中”“稳赚”或收益承诺 |
| AI 辅助整理的研究文案（含风险提示） | 官方赛果替代、内幕信息或交易通道 |
| 合规声明、AI 声明与隐私最小化说明 | 个人身份信息采集与广告画像 |

## 3. 治理原则

1. **研究优先**：内容用于体育数据分析学习、模型研究与复盘讨论。  
2. **透明披露**：涉及 AI 辅助生成的内容将明确提示。  
3. **责任边界**：用户须遵守所在地法律法规，并自行承担其行为后果。  
4. **隐私最小化**：平台以静态展示为主，不主动收集投注账户、支付或证件信息。  
5. **可审计表达**：复盘指标按层级拆分，避免单一“命中率”掩盖模型偏差。

## 4. 风险摘要（请完整阅读合规文件）

- 模型估计与赛果可能显著偏离；  
- AI 内容可能错误、滞后或不完整；  
- 本平台输出**不构成**投注、购彩、理财或投资建议；  
- 用户须自行遵守当地法律并承担自身行为责任。
