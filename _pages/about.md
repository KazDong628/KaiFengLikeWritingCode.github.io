## 2. 产品定位

| 维度 | 说明 |
| --- | --- |
| 产品名称 | MatchStruct / 赛理 |
| 产品类型 | 结构数据分析研究演示平台（静态信息展示） |
| 核心能力 | 分层复盘、情景推演、方法透明、合规提示、多品类切换 |
| 不适用场景 | 投注/购彩指导、收益承诺、跟单代购、资金往来 |

## 3. 我们提供 / 不提供

| 我们提供 | 我们不提供 |
| --- | --- |
| 基于公开信息的结构分析与情景覆盖（足球已演示） | 投注建议、购彩指导、跟单或代购 |
| 待开发模块的路线图与验收门槛披露 | “必中”“稳赚”或收益承诺 |
| AI 辅助整理的研究文案（含风险提示） | 官方开奖替代、内幕信息或交易通道 |
| 合规声明、AI 声明与隐私最小化说明 | 个人身份信息采集与广告画像 |

## 4. 治理原则

1. **研究优先**：用于数据分析学习、模型研究与复盘讨论。  
2. **分模块门禁**：彩票类模块须通过独立回测验证后，方可展示研究输出。  
3. **透明披露**：AI 辅助内容明确提示；待开发状态清晰标注。  
4. **责任边界**：用户须遵守所在地法律法规，并自行承担其行为后果。  
5. **隐私最小化**：不主动收集投注账户、支付或证件信息。
