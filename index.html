      模型迭代结论

      周二复盘后核心：限制低概率单选，让球反向约束普通赛果，补齐高低尾与联合分布一致性。

      01单选门控：最高概率<48%禁止单胆；一二差<12pt强制双选；平≥29%须进入主要方向
      02让球联动：-1让负≥57%禁止主胜单选；+1让胜≥57%须标受让不败；冲突自动降置信度
      03总球三状态：封闭(0-1) / 常规 / 崩溃(4+)；平局高且进攻弱时优先0—2球并纳入0-0
      04欧战节奏差与防线崩溃模块；首回合抬1-0；防负式三覆盖不计有效防守命中

      研究节奏

      从样本建设到公开口径的迭代顺序（演示）。

      Phase A
      口径冻结统一 90′ 竞彩口径；淘汰赛晋级与比分拆分记账。
      Phase B
      分层复盘按赛事类型输出 KPI + 亮点/失误场 + 逐场明细。
      Phase C
      排序器在七比分候选生成后增加二阶段排序，压低“池内但非前三”。
      Phase D
      公开验收同时披露核心3 / 完整池 / 方向 / 球数，避免多选覆盖被误读为单点命中。

      口径说明

      • 胜平负主选：只统计第一推荐；覆盖含明确防平/双选。「防负」式三覆盖不计有效防守命中。
      • 让球主选 / 范围：第一推荐或明确第二防守方向。
      • 总进球：核心区间及明确防守球数。
      • 七比分：90分钟准确比分是否进入七比分池。
      • 世界杯决赛西班牙—阿根廷按 90 分钟 0-0 统计（加时另计）。

      术语速查

      核心3真实比分落在前三个候选比分。
      比分池落在完整七比分候选，但不在前三。
      方向命中精确比分未中，但胜平负方向正确。
      防守命中第一选择未中，明确第二/防平方向兑现。
